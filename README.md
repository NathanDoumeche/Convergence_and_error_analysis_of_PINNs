# Convergence_and_error_analysis_of_PINNs
Convergence and error analysis of PINNs

## Install

Python 3.9.16

    pip3 install -r requirements.txt

## Run figures

    cd Figures/
    python3 prop3-1.py
    python3 prop3-2.py

This creates Figure 3.1 and 3.2 in the file Figures/Outputs.

## Run experiments

    cd Experiments/
    python3 PINNs_2d.py

This creates ... in the file Experiments/Outputs.
